Images

Pure CSS's fluid image class serves as our default for image styles.

Markup:


<img class="pure-img" src="http://www.modernhealthcare.com/images/pharmaceuticals.jpg" alt="Pharmaceuticals">

Styles:


.pure-img {
    max-width: 100%;
    height: auto;
    display: block;
}

Picturefill by Filament Group is our clientside JS solution for responsive images.

Markup:


<picture>
  <!--[if IE 9]><video style="display: none;"><![endif]-->
  <source srcset="http://placehold.it/1280x960&text=XL" media="(min-width: 1024px)">
  <source srcset="http://placehold.it/1024x768&text=LG" media="(min-width: 768px)">
  <source srcset="http://placehold.it/960x720&text=MD" media="(min-width: 568px)">
  <!--[if IE 9]></video><![endif]-->
  <img class="pure-img" srcset="http://placehold.it/768x576&text=SM" alt="<%Title%>">
</picture>
